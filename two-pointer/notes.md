# Two Pointer Method

### When to use

- Array / string
- Need pairs, triplets, subarrays
- Sorted array or can be sorted

### Core idea

Use **two indices** that move based on conditions instead of nested loops.

### Variants

- Opposite ends (`l`, `r`)
- Same direction (slow–fast)
- Fixed + moving pointer

### Common tricks

- Sorting often enables two pointers
- Be careful with duplicates
- Update answer **before** moving pointers

### Typical problems

- Two Sum (sorted)
- Remove duplicates
- Container With Most Water
- Valid Palindrome
